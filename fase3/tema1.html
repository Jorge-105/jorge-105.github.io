Etiquetas basicas de html

HTML5 es la version de HTML que corresponde a las siglas en ingles (Hiper Text Markup Language o Lenguajes de marca de hipertexto, este lenguaje de programación usa una serie de codigos llamadas etiquetas, estas definen los elementos que componen la pagina web como imagenes, texto etc. los navegadores de internet con Explorer son las encargados de interpretar estas etiquetas y ser mostradas al usuario en la pagina web.
